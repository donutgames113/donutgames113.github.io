Drop your songs into this folder.

Tips:

- Subfolders are fine.
- Sorting comes from the metadata inside the file, not the folder name.
- For the best result, fill in album, artist, title, track number, year, genre, and cover art before rebuilding.

After adding or changing songs, run:

npm run build:music

Then open:

media/index.html

or:
npm.cmd run watch:music
